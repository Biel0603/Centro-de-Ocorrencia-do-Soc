# SOC Insight

O **SOC Insight** é uma plataforma web desenvolvida para otimizar as operações de um **Security Operations Center (SOC)** em centros de distribuição. O sistema centraliza o gerenciamento de ocorrências, facilita a comunicação entre os turnos e disponibiliza dashboards com indicadores em tempo real, auxiliando operadores, analistas e gestores na tomada de decisões.

A plataforma oferece funcionalidades como registro e acompanhamento de ocorrências, histórico de eventos, passagem de turno, controle de pendências, relatórios e gráficos gerenciais, proporcionando maior organização, produtividade e eficiência operacional.

Este projeto foi desenvolvido com foco na transformação digital dos processos de segurança patrimonial, utilizando tecnologias web modernas para oferecer uma interface intuitiva, responsiva e de fácil utilização.

## Funcionalidades

* Dashboard com indicadores operacionais.
* Registro e gerenciamento de ocorrências.
* Passagem de turno com controle de pendências.
* Histórico de ocorrências com pesquisa e filtros.
* Relatórios e gráficos para análise de desempenho.
* Gerenciamento de usuários e perfis.
* Interface moderna e responsiva.

## Objetivo

O principal objetivo do SOC Insight é fornecer uma solução tecnológica capaz de centralizar informações, reduzir falhas operacionais, melhorar a comunicação entre as equipes e apoiar a tomada de decisões através de dados em tempo real.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* JSON Server (API simulada)
* Git e GitHub
